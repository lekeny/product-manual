Token Bridge is a [Cobo Portal App](/en/portal/portal-apps/introduction) maintained by Cobo that addresses the challenge of moving assets across different blockchain networks. It enables users to perform secure cross-chain transactions with their [Custodial Wallet](/en/portal/custodial-wallets/introduction) assets, eliminating the complexity and risks typically associated with manual cross-chain transfers.

## Why Use Token Bridge?

- **Simplified Cross-Chain Operations**: Eliminate the complexity of managing multiple wallets and exchanges for cross-chain transfers
- **Enhanced Security**: Leverage Cobo's institutional-grade security infrastructure and [Cobo Guard](/en/guard/introduction.mdx) approval system
- **Cost-Effective**: Optimize transaction costs through automated routing and bulk processing
- **Time-Saving**: Streamline operations with a single interface for all cross-chain needs

## How It Works

1. **Asset Selection**: Choose the source asset and destination chain/token
2. **Transaction Initiation**: Specify the amount and recipient address
3. **Security Verification**: Transaction undergoes [Cobo Guard](/en/guard/introduction.mdx) approval process
4. **Cross-Chain Transfer**: Automated handling of the cross-chain conversion and transfer
5. **Confirmation**: Real-time tracking and verification of transaction status

## Key Features

- **Cross-chain Transfers**:
  - Move assets seamlessly between supported blockchain networks
  - Support for major networks including Ethereum, BSC, Polygon, and more
  - Real-time exchange rate information

- **Token Conversion**:
  - Convert between native tokens and their wrapped versions
  - Automatic handling of intermediate conversion steps
  - Competitive conversion rates

- **Transaction Monitoring**:
  - Real-time status tracking
  - Detailed transaction progress indicators
  - Email notifications for important status updates

- **Security Features**:
  - [Cobo Guard](/en/guard/introduction.mdx) multi-signature approval system
  - Risk assessment for each transaction
  - Customizable approval workflows

- **Transaction History**:
  - Comprehensive transaction records
  - Exportable reports for accounting
  - Detailed fee breakdown

## Getting Started

To start using Token Bridge, you'll need:

1. A Cobo Custodial Wallet account
2. Sufficient balance in your source wallet
3. [Cobo Guard](/en/guard/introduction.mdx) set up for transaction approval

Visit our [Quick Start Guide](/en/apps/bridge/quick-start) to begin your first cross-chain transfer.
