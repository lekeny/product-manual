## Callback Endpoint 管理功能

只需点击相应的按钮或图标，即可在开发者控制台中管理 Callback Endpoint：

- **注册 Callback Endpoint**：注册 Callback Endpoint，用于接收来自 Cobo Portal 的消息。
- **编辑** <img src="/cn/images/developer-console/icon_edit.svg" className="icon" alt="编辑图标"/>：修改所选 Callback URL 的相关字段。
- **失效** <img src="/cn/images/developer-console/icon_revoke.svg" className="icon" alt="失效图标"/>：永久禁用向所选 Callback URL 发送所有 Cobo Portal 系统通知。
- **筛选** <img src="/cn/images/developer-console/icon_filter.svg" className="icon" alt="筛选图标"/>：根据事件的状态和类型定制视图。
- **搜索** <img src="/cn/images/developer-console/icon_search.svg" className="icon" alt="搜索图标"/>：通过搜索 URL 或事件描述，快速找到 Callback Endpoint。
